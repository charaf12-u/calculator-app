# Calculator App 📱

A simple and intuitive calculator application built with Flutter.

## Features ✨
- **Basic Operations:** Addition (+), Subtraction (-), Multiplication (*), and Division (/).
- **Error Handling:** Protects against division by zero.
- **Clean UI:** Easy to use interface with clear result displays.

## Getting Started 🚀

This project is a starting point for a Flutter application.
To run this project locally:

1. Make sure you have [Flutter installed](https://docs.flutter.dev/get-started/install).
2. Clone this repository:
   ```bash
   git clone https://github.com/charaf12-u/calculator-app.git
   ```
3. Navigate to the directory:
   ```bash
   cd calculator-app
   ```
4. Install dependencies:
   ```bash
   flutter pub get
   ```
5. Run the app:
   ```bash
   flutter run
   ```
